# 基于single-spa-angular封装的angular7子应用

使用single-spa-angular:build作为builder启动,监听端口4208

参考：https://zh-hans.single-spa.js.org/docs/ecosystem-angular#use-angular-builder

# 运行步骤：

* cd child-my-app
* npm install
* npm run serve:single-spa:my-app
